PieShop

Mar, 2018 - Mar, 2018

ASP .NET Core 2 MVC, Identity, EntityFramework Core, Javascript, JQuery, Gulp, Postgres, HTML5, CSS3

Sources: https://github.com/benthinnick/PieShop

Веб-приложение вымышленной пекарни, сделано на основе двух курсов Pluralsight. Построено на фреймворке ASP .NET Core 2.0 MVC. Приложение дает возможность заказывать пироги, оставлять рецензии, имеет разделение пользователей по ролям (администратор, пользователь). Использовал Entity Framework Core для связи сущностей с базой данных. Postgres для баз данных, Serilog как логгер, JQuery для client-side, xUnit для тестов и сам фреймворк для валидации моделей, кэширования и ауторизации.
